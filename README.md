# IPL Score Prediction

This project predicts IPL cricket match scores using machine learning techniques based on historical match data.

## Objective
To build a predictive model that estimates the final score of an IPL match based on match conditions and team performance.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Project Workflow
1. Data collection and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature engineering
4. Model training using Random Forest Regression
5. Model evaluation and prediction

## Dataset
The dataset contains historical IPL match data used to train the model.

## Outcome
The trained machine learning model predicts IPL match scores with reasonable accuracy using match features such as overs, wickets, and current score.
